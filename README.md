# HPC-NRW Quick Start Guides

HPC.NRW is the North Rhine-Westphalia state initiative for high-performance computing, aiming to support researchers by providing easy-to-follow Quick Start Guides for running workloads on HPC systems.
It contains multiple guides, such as Distributed Data Parallel (DDP) with PyTorch, to help new users get started quickly on HPC clusters.

### Available Guides 
[**DDP Guide**](DDP/) : Training a CNN on FashionMNIST using PyTorch Distributed Data Parallel (DDP) on HPC.
